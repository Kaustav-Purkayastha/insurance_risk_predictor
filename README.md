# Data-Driven Risk Analysis and Claim Prediction in Insurance using Open-Source Technologies

**Author:** Kaustav Purkayastha — LPU MCA Final Year  
**Role:** Data Engineer — Deloitte

## Project Description
This project analyzes insurance customer and policy data to identify risk patterns and predict premium charges using machine learning. It demonstrates EDA, feature engineering, SQL integration, and model deployment using Python.

## Dataset
Public insurance dataset: [insurance.csv](https://raw.githubusercontent.com/stedy/Machine-Learning-with-R-datasets/master/insurance.csv)

## Folder Structure
- `data/` : raw CSV dataset  
- `outputs/figures/` : charts and plots (PNG)  
- `outputs/models/` : saved ML model (`insurance_model.pkl`)  

## How to Run
1. Open Jupyter Notebook in this folder.  
2. Run `01_insurance_claim_analysis.ipynb` top to bottom.  
3. All outputs (figures, model) will be saved automatically in the `outputs` folder.
